### 6.3.1 文档字符串（docstring）

    放在函数开头，将作为函数的一部分存储起来

例如：

```python
def test_fun():
    'this is docstring'
    return 0
```
访问docstring的方法:

1、

```python
test_fun._doc_
```

2、

```python
help(test_fun)
```

> 所有的函数都有返回值。如果你没有告诉它们该返回什么，将返回None
