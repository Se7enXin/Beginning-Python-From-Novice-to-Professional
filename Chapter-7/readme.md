### 7.2.4 再谈隐藏

要让方法或属性成为私有的（不能从外部访问），只需让其名称以两个下划线打头即可

```python
class Secretive:
    def __inaccessible(self):
        print("Bet you can't see me ...")

    def accessible(self):
        print("The secret message is:")
        self.__inaccessible()
```
